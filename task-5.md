> ## psedo code for the armstrong number:

---
```
READ n
temp=n
sum=0
WHILE n>=0
    sum=sum+(n%10)*(n%10)*(n%10)
    n=n/10
ENDWHILE

IF sum==temp
    WRITE "NUMBER IS AN ARMSTRONG NUMBER"
ELSE
    WRITE "NUMBER IS NOT AN ARMSTRONG NUMBER"
 ```   
